# Castaway - Weather Dashboard

A modern, responsive weather dashboard that provides real-time weather information using a weather API.

## 🌐 Live Demo
https://pranav09130.github.io/Castaway/

## 📂 GitHub Repository
https://github.com/Pranav09130/Castaway

## ✨ Features
- Real-time weather updates
- City search
- Interactive weather map
- Air Quality Index
- 5-day weather forecast
- Weather alerts
- Responsive design

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript
- Weather API
- OpenStreetMap
- Chart.js

## 🚀 How to Run
1. Clone the repository.
2. Open `index.html` in your browser.
3. Or visit the live demo using the link above.
